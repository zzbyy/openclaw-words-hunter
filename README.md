# Words Hunter — OpenClaw plugin

OpenClaw extension for [Words Hunter](https://github.com/zzbyy/words-hunter): vocabulary tools for Obsidian vaults (scan, load word, record mastery, etc.).

## Install

```bash
openclaw plugins install github:zzbyy/openclaw-words-hunter
```

Add the plugin id to `plugins.allow` in OpenClaw config if you use an allowlist.

## Update

```bash
openclaw plugins update words-hunter
```

## Development

```bash
npm install
npm run build
npm test
```
